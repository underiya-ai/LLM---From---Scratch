# Step-01: Data Cleaning

## Objective
Clean raw text data for training a Language Model (LLM).

---

## 📂 Files

- the-verdict.txt → Original raw dataset
- cleaned.txt → Cleaned output
- data_cleaning.py → Python cleaning code

---

## Cleaning Steps

- Converted text to lowercase
- Removed HTML tags
- Removed URLs
- Removed numbers
- Removed special characters
- Normalized spaces

---

##  How to Run

```bash
python data_cleaning.py
