# 🧠 LLM From Scratch (Step-by-Step)

This repository contains my complete journey of building a Large Language Model (LLM) from scratch.

##  Goal
To deeply understand how LLMs work internally by implementing each component step-by-step without using prebuilt models.

---

## 📂 Project Structure

- Step 01: Data Processing
- Step 02: Tokenization
- Step 03: Embeddings
- Step 04: Attention Mechanism
- Step 05: Transformer Architecture
- Step 06: Training
- Step 07: Inference

---

## What I am Learning

- How text is converted into tokens
- How embeddings work
- Attention mechanism (Self-Attention, Multi-head Attention)
- Transformer architecture (Encoder/Decoder)
- Training LLM from scratch
- Text generation

---
